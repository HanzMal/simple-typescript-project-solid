# simple-typescript-project-solid

Simple project with SOLID Principle in Typescript. The goal is can manage project Typescript and implement SOLID principle

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### 1. Fork the Repository

Click the **Fork** button at the top-right corner of this page to create a copy of this repository under your GitHub account.

### 2. Clone and Open in VS Code

Clone your forked repository and open the project folder in VS Code:

### bash

```bash
git clone "https://github.com/HanzMal/simple-typescript-project-solid.git"
cd simple-typescript-project-solid
code .
```

### 3. Install Dependency

```bash
npm install
```

### 4. run

```bash
npm run dev
```
