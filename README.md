# 🚀 Cara Menjalankan Project

Project ini sekarang menggunakan **Makefile** untuk memudahkan eksekusi:

1. **Install Dependencies:** `npm install`
2. **Build & Run:** `make`
3. **Run Only:** `make run`
4. **Clean Build:** `make clean`

## 🛠️ Tech Stack

- **Language:** TypeScript
- **Runtime:** Node.js (v20.12.0+)
- **Database:** SQLite (via better-sqlite3)
- **Environment:** Linux / WSL Ubuntu
